MyNewVoice v124 - Settings fullscreen real screen

Built from v123.

Purpose:
- Apply the same real-screen approach to the Settings dashboard itself.
- Remove the remaining floating/centred Settings panel behaviour shown in v123.

Changes:
- Settings overlay now fills the full app viewport as a real screen.
- Settings panel no longer uses max-width floating modal layout, rounded outer window, or translucent backdrop.
- Main communication container is visually hidden while Settings is open and restored on Return to App.
- Settings header remains fixed and dashboard content scrolls internally.
- Content Editor remains the v123 real-screen table editor.

Preserved:
- Dad-facing communication screen content and layout after returning to app.
- Speech behaviour and fallback.
- Local photos and recorded voices.
- Backup/export/import schema and protected top-level fields.
- Phrase IDs, category IDs, media IDs and storage keys.
