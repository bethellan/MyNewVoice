MyNewVoice v48 - Modern material themes and settings arrow cleanup

Changes:
- Replaced the previous novelty theme list with modern material-style themes:
  Default, Metallic, Photoreal, Realistic, Marble, Metal, Water.
- Reworked theme colours and surfaces to give clearer modern/realistic material effects while preserving the Dad-facing layout.
- Added subtle CSS-only material surface effects for metal, marble and water styles.
- Removed the chevron/greater-than symbol from the first Settings entry menu Information and Settings buttons.

Preserved:
- Backup/export/import schema unchanged.
- Existing protected top-level backup fields unchanged.
- Phrase IDs, category IDs, media IDs and storage keys unchanged.
- Local images/audio unchanged.
- Speech fallback unchanged.
- Dad-facing phrase row structure, button sizes and scrolling behaviour unchanged.

Compatibility:
- Theme preference remains stored in optional appSettings.theme.
- Old v46/v47 theme names that are no longer offered will fall back to Default when settings are normalised.
