Optional zoom images

Place larger display images here if you want a phrase click to briefly show a large image.

Filename rule:
- Use the phrase ID as the filename.
- Preferred format: JPG.
- Examples: food01.jpg, health03.jpg, people01.jpg, comfort12.jpg

Behaviour:
- When the phrase is tapped, the app speaks the words aloud.
- If a matching image exists in this folder, it displays for 4 seconds.
- If no matching image exists, nothing extra appears and the app works normally.

Supported extensions tried by the app: .jpg, .jpeg, .png, .webp
