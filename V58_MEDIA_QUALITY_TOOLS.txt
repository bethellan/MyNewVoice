MyNewVoice v58 - Media Quality Tools

Adds a Storage section in Settings.

New tools:
- Check size: shows local image and audio storage totals.
- Compress images: recompresses local saved images using a good-quality target after confirmation.
- Make smaller: recompresses local saved images more aggressively after confirmation.

The tool works on local saved image records only. It does not automatically alter audio recordings. Export a complete backup after compressing and checking the result.

Preserved:
- Backup/export/import schema unchanged.
- Protected top-level backup fields unchanged.
- Phrase/category/media IDs unchanged.
- Storage keys unchanged.
- Local audio unchanged.
- Speech fallback unchanged.
- Dad-facing menu and phrase row structure unchanged.
- v55 simplified Settings/About/offline status retained.
- v56 My People relationship dropdown retained.
- v57 media size display retained.
