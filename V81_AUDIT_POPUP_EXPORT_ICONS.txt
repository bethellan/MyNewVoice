MyNewVoice v81 - Audit, popup close mode, export destination handling and expanded fallback icons

Based on v80.

Changes:
- Added Settings > Voice > Message close behaviour with Timed close or Press anywhere to close.
- Press-anywhere mode keeps the phrase/image popup visible until the user taps anywhere on the popup overlay.
- Export complete backup now tries Save As picker where supported, then iOS/iPadOS share sheet where supported, then standard browser download fallback.
- Expanded offline-safe fallback icon chooser with a larger curated emoji set covering comfort, health, food/drink, self-care, activities, people, outdoors, travel and hobbies.
- Updated app/cache/version references to v81.

Preserved:
- Complete backup schema fields unchanged.
- Phrase IDs, category IDs, media IDs and storage keys unchanged.
- Local IndexedDB images/audio unchanged.
- Speech fallback behaviour unchanged.
- Existing menu and phrase content unchanged.
