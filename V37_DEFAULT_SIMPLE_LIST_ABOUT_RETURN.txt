MyNewVoice v37 - default simple list and About return behaviour

Changes:
- Simple Vocabulary View is now the default display mode for new installs/devices with no saved display preference.
- The About box now opens as a Settings sub-screen. Closing About returns to the Settings menu instead of returning directly to the app.
- Return to App remains the explicit way to leave Settings from the Settings main menu.
- Refresh app code remains allowed to leave Settings because the app reloads.

Unchanged:
- Dad-facing phrase/button behaviour.
- Backup/export/import schema and protected fields.
- Local images/audio storage.
- Phrase IDs, category IDs, media IDs and storage keys.
