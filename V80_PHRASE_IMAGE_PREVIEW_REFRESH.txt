MyNewVoice v80 - Phrase image preview refresh fix

Purpose:
- Fixes a timing issue where a newly added local phrase photo could appear in the enlarged speak popup but not appear in the small submenu row preview until after the popup closed.

Changes:
- Local phrase/menu images are set to eager loading when private media is resolved.
- The row image element is shown immediately once local media exists, while the fallback icon remains hidden to prevent fallback-icon flash.
- Handles mobile cached-image timing by using onload, complete/naturalWidth, and decode() where available.

Preserved:
- Backup/export schema unchanged.
- Phrase/category/media IDs unchanged.
- Local media storage keys unchanged.
- Speech and recorded voice fallback unchanged.
- v79 iPhone save/cancel footer audit retained.
- v78 main-menu image flash fix retained.
